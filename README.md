# Ótica Du & Edu - Landing Page

Landing page em React + TypeScript com Tailwind CSS para apresentar a Ótica Du & Edu, suas marcas premium e dicas rápidas
para entender a receita oftalmológica.

## Como executar

```bash
npm install
npm run dev
```

Crie um arquivo `.env` caso precise adicionar chaves ou URLs externas adicionais.
