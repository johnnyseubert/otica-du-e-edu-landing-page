# Ótica Du & Edu - Landing Page

Landing page em React + TypeScript com Tailwind CSS para apresentar a Ótica Du & Edu, sua história, marcas parceiras e um guia
para entender a receita de óculos.

## Tecnologias
- Vite + React + TypeScript
- Tailwind CSS com utilitários customizados
- clsx para composição de classes
- ESLint para padronização de código

## Como rodar
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o ambiente de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Acesse em `http://localhost:5173`.

## Estrutura de pastas
- `src/App.tsx`: layout principal da landing page.
- `src/data/`: textos e pontos fortes das marcas e termos de prescrição.
- `src/index.css`: estilos globais e utilitários Tailwind.

## Conteúdo
- Hero com informações de contato e destaque dos 15 anos de atuação.
- Seção sobre a história e localização (Rua Almirante Barroso, Nº 45, sala 3, Vila Nova, Blumenau - CEP 89035-402).
- Destaque das marcas Varilux, Kodak, Prime, Optview, Zeiss e Hoya com pontos positivos de cada uma.
- Guia simples para entender esférico, cilíndrico, eixo e adição na receita.
- Chamada para agendar atendimento por telefone, WhatsApp ou e-mail.
