# Ótica Du & Edu — Landing Page

Landing page criada com React + TypeScript + Vite e estilizada com Tailwind CSS e clsx.

## Scripts

- `npm install` — instala dependências.
- `npm run dev` — roda o servidor de desenvolvimento.
- `npm run build` — cria a build de produção.
- `npm run preview` — serve a build gerada.
- `npm run lint` — executa o ESLint.

## Conteúdo

A página apresenta:
- História da ótica, destaques de atendimento e variedade.
- Seção com marcas de lentes (Varilux, Kodak, Prime, Optview, Zeiss, Hoya) e seus diferenciais.
- Guia simples para interpretar uma receita de óculos: esférico, cilíndrico, eixo e adição.
- Informações de contato, endereço e formas de pagamento.
